# imaTime

A minimal, elegant time application inspired by Japanese culture. imaTime provides a clean interface for viewing the current time across different timezones with customizable themes.

## Features

- 🕒 Real-time clock with seconds toggle
- 🌍 Timezone selection
- 🎨 Multiple themes (OLED, Blue, Matrix)
- 📱 Minimal mode for distraction-free viewing
- 🇯🇵 Japanese-inspired design philosophy

## About the Name

"Ima" (今) means "now" in Japanese, reflecting the present moment. The name honors the Japanese concept of "いまここ" (ima-koko) - the here and now - emphasizing mindful awareness of the present moment.

## Technologies

- React
- TypeScript
- Tailwind CSS
- Lucide Icons
